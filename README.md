# CHIP-8 Emulator (Terminal + NodeJS)

## Usage:
1. Download this repository.
2. Install NodeJS.
3. Add your rom-file into the 'ROM' folder.
4. Navigate to the CHIP-8 folder in your terminal.
5. type: node main.js ./ROM/'insert name of romfile here'
6. Tip is to use the tab key after typing ./ROM/ to cycle through different rom-files.
7. All done! Look at controls below.

### Controls
Use these keys on a QWERTY keyboard for controls.
```
1  2  3  4            1  2  3  C 
Q  W  E  R            4  5  6  D 
A  S  D  F   ------>  7  8  9  E  
Z  X  C  V            A  0  B  F 
  qwerty                chip-8
```
And type Ctrl+C to exit.
