# CHIP-8 Emulator (Terminal + NodeJS)
- Works on linux.

## Usage:
1. Download this repository.
2. Install NodeJS.
3. Navigate to the CHIP-8 folder in your terminal.
4. type in terminal: node src/main.js
5. Choose between pre-existing ROMs and change colors in settings.
6. All done! Look at controls below.

If it scales weird in terminal, try zooming in or out.

### Controls
Use these keys on a QWERTY keyboard for controls.
```
1  2  3  4            1  2  3  C 
Q  W  E  R            4  5  6  D 
A  S  D  F   ------>  7  8  9  E  
Z  X  C  V            A  0  B  F 
  qwerty                chip-8
```
And type Ctrl+C to exit.
